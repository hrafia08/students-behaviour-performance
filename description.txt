
Streamlit Application: Student Success Predictor

Features Used:
1. Age
2. Study Hours Per Day
3. Attendance (%)
4. GPA
5. Motivation (Scale 1 to 5)
6. Cluster Average GPA

User Interface:
- Numeric input for age (10 to 25)
- Slider for study hours (0.0 to 10.0)
- Slider for attendance percentage (0% to 100%)
- GPA input (0.0 to 4.0 scale)
- Motivation scale input (1 to 5)
- Input for average GPA of the cluster (0.0 to 4.0)

Output:
- Predict button triggers the model.
- Displays predicted result (classification label).
